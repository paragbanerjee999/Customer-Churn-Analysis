# 📉 Customer Churn Analysis Dashboard

## 🔍 Overview
Analyzed 7,032 telecom customer records to identify
churn drivers and built a machine learning model to
predict customer churn with 79.2% accuracy.

## 🛠 Tools Used
- Python 3.14
- Pandas, NumPy, Seaborn, Matplotlib
- Scikit-learn (Logistic Regression)
- Power BI (Interactive Dashboard)
- Jupyter Notebook

## 📊 Dashboard Preview
![Dashboard](dashboard_page1.png)

## 📈 Key Results
- Model Accuracy: 79.2%
- Overall Churn Rate: 26.58%
- Total Records Analyzed: 7,032
- Month-to-month contracts: Highest churn risk
- Fiber optic customers churn more than DSL users
- Senior citizens show higher churn tendency

## 🗂 Project Structure
- churn_analysis.ipynb — Complete Python analysis
- churn_clean.xlsx — Cleaned dataset (7,032 rows)
- dashboard_page1.png — Power BI Dashboard screenshot
- churn_analysis.html — Notebook in HTML format

## 💡 Key Insights
1. Month-to-month contract customers churn 3x more
than two-year contract customers
2. Fiber optic internet service has highest churn
3. Higher monthly charges = higher churn probability
4. Senior citizens churn at a higher rate

## 🤖 Model Details
- Algorithm: Logistic Regression
- Training data: 80% (5,625 records)
- Testing data: 20% (1,407 records)
- Accuracy: 79.2%
- Precision (Stayed): 0.84
- Precision (Churned): 0.63
